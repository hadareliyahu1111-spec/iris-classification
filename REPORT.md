# דוח סיווג פרחי האירוס — 4 מחלקות (כולל iris-fake)

---

## 1. תיאור מערך הנתונים

מערך הנתונים המורחב מבוסס על **Iris** הקלאסי מ-`sklearn.datasets`, ומכיל **200 דגימות** ב-**4 מחלקות**:

| מחלקה | סוג | מספר דגימות |
|--------|------|--------------|
| Iris Setosa | אמיתי | 50 |
| Iris Versicolor | אמיתי | 50 |
| Iris Virginica | אמיתי | 50 |
| **Iris-Fake** | **סינתטי (AI)** | **50** |

### מחלקת iris-fake — פרח דמיוני שנוצר על ידי AI

המחלקה הרביעית נוצרה באמצעות:
1. חישוב **ממוצע התכונות** של 150 הדגימות האמיתיות
2. הוספת **רעש גאוסיאני** בסטיית תקן של 50% מהסטייה האמיתית
3. `random_state=42` לשחזוריות מלאה

כל דגימה מכילה **4 תכונות** מספריות:

| תכונה | תיאור |
|--------|--------|
| `sepal_length` | אורך גביע (ס"מ) |
| `sepal_width` | רוחב גביע (ס"מ) |
| `petal_length` | אורך עלה (ס"מ) |
| `petal_width` | רוחב עלה (ס"מ) |

## 2. תיאור המודל — MLPClassifier

### ארכיטקטורת הרשת העצבית:

```
קלט (4 נוירונים)
    ↓
שכבה נסתרת 1 — 64 נוירונים [ReLU]
    ↓
שכבה נסתרת 2 — 32 נוירונים [ReLU]
    ↓
פלט (4 נוירונים — מחלקה אחת לכל סוג)
```

### פרמטרי המודל:

| פרמטר | ערך |
|--------|-----|
| `hidden_layer_sizes` | (64, 32) |
| `activation` | relu |
| `solver` | adam |
| `max_iter` | 300 |
| `random_state` | 42 |
| `test_size` | 0.2 (40 דגימות) |
| `train_size` | 0.8 (160 דגימות) |

1. **ביצועים טובים:** דיוק של **90%** על מערך 4 מחלקות, כולל מחלקה סינתטית
2. **בלבול צפוי:** iris-fake מבולבלת עם versicolor בגלל תכונות ממוצעות דומות
3. **Setosa ו-Virginica** — זוהו בצורה מושלמת בזכות תכונות ייחודיות יותר
4. **אזהרת התכנסות:** הגדלת `max_iter` ל-500+ תשפר את האימון
5. **המלצה:** הוספת Cross-Validation ו-Precision/Recall/F1 לניתוח מעמיק יותר
