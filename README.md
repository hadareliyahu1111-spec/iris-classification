## 🌺 מערך הנתונים — Iris Dataset

מערך הנתונים **Iris** הוא אחד ממערכי הנתונים המפורסמים ביותר בהיסטוריה של למידת מכונה.
פורסם לראשונה על ידי הביולוג והסטטיסטיקאי **רונלד פישר** בשנת **1936**.

### 🔬 שלושת המינים

| מין | שם מדעי | מאפיינים |
|-----|----------|-----------|
| 🔴 Setosa | *Iris setosa* | עלים קצרים וצרים מאוד — קל לזיהוי |
| 🟢 Versicolor | *Iris versicolor* | ממוצע בין השניים — חפיפה עם Virginica |
| 🔵 Virginica | *Iris virginica* | עלים ארוכים ורחבים — הגדולה מבין השלוש |

### 📐 ארבע התכונות (Features)

| תכונה | שם באנגלית | תיאור |
|--------|-----------|-------|
| אורך גביע | Sepal Length | אורך עלי הכותרת החיצוניים (ס"מ) |
| רוחב גביע | Sepal Width | רוחב עלי הכותרת החיצוניים (ס"מ) |
| אורך עלה | Petal Length | אורך עלי הכותרת הפנימיים (ס"מ) |
| רוחב עלה | Petal Width | רוחב עלי הכותרת הפנימיים (ס"מ) |

**סה"כ:** 150 דגימות — 50 לכל מין.

## 🧠 המודל — MLPClassifier

**MLP (Multi-Layer Perceptron)** היא רשת עצבית מלאכותית הבנויה משכבות של נוירונים מחוברים זה לזה.
הרשת לומדת לזהות דפוסים בנתונים באמצעות התאמת משקולות (weights) דרך תהליך חזרתי הנקרא **Backpropagation**.

### 🏗️ ארכיטקטורת הרשת

```
קלט (4 נוירונים)
    ↓
שכבה נסתרת 1 — 64 נוירונים [ReLU]
    ↓
שכבה נסתרת 2 — 32 נוירונים [ReLU]
    ↓
פלט (3 נוירונים — מחלקה אחת לכל מין)
```

### ⚙️ פרמטרי המודל

| פרמטר | ערך | הסבר |
|--------|-----|-------|
| `hidden_layer_sizes` | `(64, 32)` | שתי שכבות נסתרות |
| `activation` | `relu` | פונקציית הפעלה — מאפשרת למידת דפוסים מורכבים |
| `solver` | `adam` | אופטימייזר — מעדכן משקולות ביעילות |
| `max_iter` | `300` | מספר איטרציות מקסימלי |
| `random_state` | `42` | קיבוע לשחזוריות |

### 📊 חלוקת הנתונים

```
150 דגימות סה"כ
├── 🏋️ אימון (Train): 120 דגימות — 80%
└── 🧪 בדיקה (Test):   30 דגימות — 20%
```
